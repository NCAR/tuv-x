###########
Citing TUV-x
###########

TUV-x is a software library built on a body of scientific work.
Please cite both the science and the software.

Citing the Science
==================

The scientific foundation of TUV-x is described in:

.. code-block:: bibtex
   :caption: Madronich & Flocke (1999)

   @incollection{Madronich1999,
     author    = {Madronich, Sasha and Flocke, Siri},
     title     = {The Role of Solar Radiation in Atmospheric Chemistry},
     booktitle = {Environmental Photochemistry},
     series    = {The Handbook of Environmental Chemistry},
     editor    = {Boule, Pierre},
     publisher = {Springer},
     address   = {Berlin, Heidelberg},
     pages     = {1--26},
     year      = {1999},
     doi       = {10.1007/978-3-540-69044-3_1},
     url       = {https://doi.org/10.1007/978-3-540-69044-3_1},
   }

Citing the Software
===================

We use `Zenodo <https://doi.org/10.5281/zenodo.7126039>`_ to associate a DOI
with each release of TUV-x.

.. code-block:: bibtex
   :caption: All versions

   @software{Shores2022,
     author    = {Walters, Stacy and Dawson, Matt and Shores, Kyle},
     title     = {{NCAR/tuv-x}},
     year      = {2022},
     publisher = {Zenodo},
     doi       = {10.5281/zenodo.7126039},
     url       = {https://doi.org/10.5281/zenodo.7126039},
   }
